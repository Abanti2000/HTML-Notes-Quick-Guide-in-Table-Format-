HTML Notes (Quick Guide in Table Format)

1️⃣ Basic Structure (HTML Document)
Tag Use (Purpose) Position (Where to Write?) Example
<html> Main HTML document container Starts first (root tag) <html lang="eng">...<html>
<head> Metadata, title, CSS, JS Inside <html>, before <head><title>My Page><title><head>
<body> Main content of the page Inside <html>, after <head> <body>Content here<body>
;

2️⃣ Text Elements (Headings, Paragraphs, Lists, Links, Images)
Tag Use (Purpose) Attributes (Options) Example
<h1>-<h6> Headings (Titles) <h1> largest, <h6> smallest <h2>My Skills</h2>
<p> Paragraph text No required attributes <p>This is a paragraph.</p>
<br> Line break No required attributes <p>Hello<br>World</p>
<ul> Unordered List (Bullets) No required attributes <ul><li>Item 1</li><li>Item 2</li></ul>
<ol> Ordered List (Numbers) No required attributes <ol><li>Item 1</li><li>Item 2</li></ol>
<li> List Item Only inside <ul> or <ol> <li>Item</li>
<a> Hyperlink (Add a link) href="URL" required <a href="https://example.com">Click Here</a>
<img> Display an image src="image.jpg", alt="desc" <img src="logo.png" alt="Company Logo">

3️⃣ Forms & Inputs (User Interaction)
Tag/Attribute Use (Purpose) Example
<form> Creates a form <form action="submit.php"></form>
<label> Label for input fields <label for="name">Enter Name:</label>
<input> User input field <input type="text" name="username">
type="text" Simple text input <input type="text" placeholder="Enter Name">
type="password" Password input (hidden text) <input type="password">
type="email" Email input with validation <input type="email">
type="number" Accepts only numbers <input type="number" min="1" max="100">
type="date" Date picker <input type="date">
type="radio" Radio button (One option) <input type="radio" name="gender" value="male">Male
type="checkbox" Checkbox (Multiple selection) <input type="checkbox" name="hobby" value="Singing"> Singing
<select> Dropdown menu <select><option>HTML</option></select>
<datalist> Auto-suggest Input Field <input list="languages"><datalist id="languages"><option value="HTML"></datalist>

4️⃣ Tables (Displaying Data)
Tag Use (Purpose) Example
<table> Creates a table <table>...</table>
<tr> Defines a table row <tr>...</tr>
<th> Table Headings <th>Heading</th>
<td> Table Data <td>Data</td>
colspan="2" Merges two columns <td colspan="2">Merged</td>
rowspan="2" Merges two rows <td rowspan="2">Merged</td>

5️⃣ Semantic Tags (Better Page Structure)
Tag Use (Purpose) Example
<header> Top Section (Title, Navbar) <header>Welcome!</header>
<nav> Navigation menu <nav><a href="#">Home</a></nav>
<section> Divides main content <section>About Me</section>
<article> Blog post or content block <article>My Story</article>
<footer> Bottom section of the page <footer>Copyright 2025</footer>

6️⃣ Multimedia (Audio, Video, Embed)
Tag Use (Purpose) Example
<audio> Plays an audio file <audio controls src="song.mp3"></audio>
<video> Plays a video file <video controls src="video.mp4"></video>
<iframe> Embeds external content <iframe src="YouTube Video URL"></iframe>

7️⃣ Correct Webpage Structure Order (HTML Layout)
Step Section (What to Write First?)
1️⃣ <html> (HTML Start)
2️⃣ <head> (Title, Meta Tags, CSS, JS)
3️⃣ <body> (Main Content)
4️⃣ <header> (Top Section)
5️⃣ <nav> (Navigation)
6️⃣ <section> (Main Page Content)
7️⃣ <article> (Detailed Content)
8️⃣ <ul> or <ol> (Lists)
9️⃣ <table>>(For structured data)
🔟 <form> (For user input)
🔹 <audio>, <video>, <iframe> (Multimedia)
🔚 <footer> (Copyright & Contact)
